# Malware Threat Simulation & Analysis (CEH Lab)

##  Overview

This project demonstrates practical implementation of malware attacks, including virus creation and Trojan-based remote access simulation using CEH lab tools.

---

## Concepts Covered

* Virus creation and system infection
* Trojan (RAT) attack simulation
* Payload execution and system impact
* Persistence mechanisms
* Backdoor access and remote control
* Keylogging and monitoring techniques

---

## Lab Activities

1. Created a virus using JPS Virus Maker tool
2. Executed the virus on a target system and observed behavior
3. Generated a Trojan using njRAT
4. Delivered payload to victim system
5. Established remote connection and controlled the system
6. Monitored system activities like file access, commands, and keystrokes

---

## Tools Used

* Windows 11 VM (Attacker)
* Windows Server VM (Victim)
* JPS Virus Maker
* njRAT Tool

---

##  Learning Outcomes

* Understood real-world malware behavior
* Learned how attackers gain system access
* Gained hands-on experience in cybersecurity labs
* Improved knowledge of attack and defense techniques
